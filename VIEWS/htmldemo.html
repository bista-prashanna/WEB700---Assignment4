Tables - HTML and CSS

Tables are a great way to organize data. In the example below, we’ll show how to create a responsive table with HTML and style it with CSS:

HTML Elements Table
Element Description Usage
<div> Defines a division or section in a document. Used to group content or apply CSS styles.
<h1> - <h6> Defines HTML headings. Used for defining headings of different levels.
<a> Defines a hyperlink. Used to create links to other pages or resources.
<p> Defines a paragraph. Used to group sentences and sections of text.
<img> Defines an image. Used to embed images in a document.
Footer: Basic HTML Element Information

Notice how we used the <caption>, <thead>, <tbody>, and <tfoot> tags to create a clear and structured table layout for our HTML elements.

HTML Lists - A Breakdown

Explore different types of lists in HTML. The following sections demonstrate ordered lists, unordered lists, and nested lists. These are often used in documentation, FAQs, and navigation menus:

Ordered List: HTML Elements

  1. <div> - Defines a division or section in a document.
  2. <p> - Defines a paragraph.
  3. <h1> to <h6> - Defines HTML headings.
  4. <a> - Defines a hyperlink.

This ordered list shows some common HTML tags in the order of their importance when building a web page.

Unordered List: CSS Properties

  • background-color - Sets the background color of an element.
  • font-size - Sets the size of the font.
  • padding - Sets the space between the content and the border of an element.
  • margin - Sets the space outside an element.

Here’s an unordered list of some fundamental CSS properties that help style HTML elements.

Nested List: JavaScript Methods

  1. Array Methods
    • push() - Adds one or more elements to the end of an array.
    • pop() - Removes the last element from an array.
    • shift() - Removes the first element from an array.

  2. String Methods
    • toUpperCase() - Converts a string to uppercase letters.
    • toLowerCase() - Converts a string to lowercase letters.
    • charAt() - Returns the character at a specified index in a string.

This nested list showcases JavaScript methods for arrays and strings, each with examples of commonly used functions.
